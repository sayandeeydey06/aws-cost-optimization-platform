# AWS Cost Optimization Agent

An AI-powered cloud cost optimization platform that analyzes AWS resources, identifies waste, estimates savings, and generates optimization recommendations.

## Tech Stack

- AWS
- Python
- FastAPI
- React
- Boto3
- LangGraph

## Features

- AWS Cost Explorer Integration
- EC2 Optimization
- S3 Cost Analysis
- RDS Optimization
- Savings Estimation
- AI Recommendations
